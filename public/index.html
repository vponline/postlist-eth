About

priority_high

To use this application you'll need MetaMask.

You'll also need some testnet Ether from Ropsten faucet.

This application allows you to create new posts which are stored in a smart contract on the Ethereum blockchain. The posts can also be updated or deleted, and you can search and read posts by their id number.
